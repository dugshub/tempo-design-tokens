# Tempo Design Tokens

A small utility library for managing design tokens across the Tempo product suite.

## Structure

```
src/
├── tokens/       # Token definitions (colors, spacing, typography)
├── transforms/   # Token format transformers (CSS, Tailwind, Figma)
└── utils/        # Shared utilities
```

## Development

```bash
bun install
bun run build
bun run test
```
